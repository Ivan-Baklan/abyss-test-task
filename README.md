# Test task tree view

Test task for Abyss company the site[https://abyss-test-task-kappa.vercel.app/]

## Functionality of the site
- Creating new nodes
- Removing nodes
- Editing text inputs in nodes
- Moving the tree on the page
- Centering the tree
- Scaling the tree 


## Available Scripts

In the project directory, you can run:

### `npm start`

### `npm run build`
